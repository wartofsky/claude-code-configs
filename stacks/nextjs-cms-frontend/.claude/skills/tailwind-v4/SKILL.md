## Critical Patterns
- Define design tokens in CSS with `@theme`.
- Prefer reusable primitives/components before excessive one-off class lists.
- Use mobile-first responsive variants.
- Preserve accessible focus, contrast, target sizes, and reduced motion.
- Keep utility order readable: layout, spacing, typography, colors, effects.
