## Checklist
- Persistent filters/search/sort in URL when useful.
- Clear loading, empty, error, partial-success, and retry states.
- Tables support keyboard access, responsive overflow, and stable row keys.
- Long-running tasks show progress, status, timestamps, stop/delete affordances, and terminal states.
- Dangerous actions require confirmation and explain irreversible impact.
- Notifications are actionable and not noisy.
